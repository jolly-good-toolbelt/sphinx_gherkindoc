Maintainer's Documentation
==========================

Scripts and utilities for maintainers of this package.

Top-level scripts:
------------------

    * ``bootstrap.sh`` - downloads and runs the python bootstrap ``env_setup.py``
      script from the ``jgt_tools`` GitHub repo,
      to set up your environment fully
      when you do not yet have ``jgt_tools`` installed in your virtual environment.


JGT Tools scripts:
------------------

This repo uses the scripts provided by `jgt_tools`_.
See linked documentation for reference.


.. _`jgt_tools`: https://jolly-good-toolbelt.github.io/jgt_tools/
